# Roseknot Events & Marketing — Digital Platform

An ultra-premium, conversion-optimized frontend platform designed for **Roseknot Events & Marketing** (based in New Delhi). This workspace blends structural luxury event design with high-impact corporate production pipelines, featuring fully fluid layouts, interactive client onboarding interfaces, and micro-animated component cards.

---

## 🚀 Key Architectural Features

- **Asymmetric Production Showcase Matrix:** Custom multi-dimensional gallery inspired by premium webflow layouts, breaking traditional grid restrictions to highlight bespoke project design lines.
- **Dynamic Variable-Spacing Contact System:** A robust validation-ready client intake pipeline optimized with responsive vertical offsets (`py-14` framework) to match baseline component grids perfectly.
- **Fluid Infinite Loop Marquee:** Glitch-free, requestAnimationFrame-driven scrolling engine that automatically pauses on hover/touch interactions for seamless commercial browsing.
- **Scroll-Driven Intersection Observer Engine:** Lightweight native JavaScript animation layer that updates element opacity and multi-axis transforms incrementally as users traverse sections.

## 🛠️ Technical Stack & Framework Frameworks

- **Core Layer:** Semantic HTML5 Structure (`index.html`)
- **Styling Matrix:** Tailwind CSS CDN Integration
- **Custom Compositing:** Explicit UI elevation layers and CSS transitions (`style.css`)
- **Animation Control Layer:** Vanilla JavaScript (ES6+) Event Handlers & Viewport Observers (`script.js`)
- **Visual & Vector Assets:** FontAwesome 6+ Kit Integrations
